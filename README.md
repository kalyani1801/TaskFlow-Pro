# TaskFlow-Pro

## Project Description

TaskFlow-Pro is a responsive task management web application built using HTML, CSS, and JavaScript. It helps users organize daily tasks efficiently by allowing them to add, complete, edit, and delete tasks through a clean and user-friendly interface.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Edit existing tasks
- Responsive design
- Dynamic task updates without page reload
- User-friendly interface
- Dark/Light theme support (if implemented)

## Technologies Used

- HTML5
- CSS3
- JavaScript

1. Clone the repository:

   ```bash

   ```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

4. Start managing your daily tasks efficiently.
